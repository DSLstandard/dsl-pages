# `dsl-pages`. Hosted on [pages.stay-in.uk](https://pages.stay-in.uk)

A static website housing a collection of tools I want to have as a single-page
application. Some pages are simply shoutouts to other open-source tools I've
used before and liked very *very* much.

**Hosted on my site https://pages.stay-in.uk!** There are no ADs,
cookie banners, donation pop-ups, trackers (unless I forgot to turn off
Cloudflare's analytics), etc. The static HTML/CSS/JS/other assets are hosted
with Cloudflare Pages.

Almost all UI logic are vibe-coded with the pi coding agent + ~~a local-run
LLM~~ a dirt-cheap DeepSeek subscription, plus some manual refactoring to
help the AI hallucinate less. Often times I manually edit the code to fix
critical coding errors when the AI keeps failing.
